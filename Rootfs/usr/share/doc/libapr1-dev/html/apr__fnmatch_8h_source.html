67 /**
68  * Try to match the string to the given pattern, return APR_SUCCESS if
69  * match, else return APR_FNM_NOMATCH. Note that there is no such thing as
70  * an illegal pattern.
71  *
72  * With all flags unset, a pattern is interpreted as such:
73  *
74  * PATTERN: Backslash followed by any character, including another
75  * backslash.<br/>
76  * MATCHES: That character exactly.
77  *
78  * <p>
79  * PATTERN: ?<br/>
80  * MATCHES: Any single character.
81  * </p>
82  *
83  * <p>
84  * PATTERN: *<br/>
85  * MATCHES: Any sequence of zero or more characters. (Note that multiple
86  * *s in a row are equivalent to one.)
87  *
88  * PATTERN: Any character other than \?*[ or a \ at the end of the pattern<br/>
89  * MATCHES: That character exactly. (Case sensitive.)
90  *
91  * PATTERN: [ followed by a class description followed by ]<br/>
92  * MATCHES: A single character described by the class description.
93  * (Never matches, if the class description reaches until the
94  * end of the string without a ].) If the first character of
95  * the class description is ^ or !, the sense of the description
96  * is reversed. The rest of the class description is a list of
97  * single characters or pairs of characters separated by -. Any
98  * of those characters can have a backslash in front of them,
99  * which is ignored; this lets you use the characters ] and -
100  * in the character class, as well as ^ and ! at the
101  * beginning. The pattern matches a single character if it
102  * is one of the listed characters or falls into one of the
103  * listed ranges (inclusive, case sensitive). Ranges with
104  * the first character larger than the second are legal but
105  * never match. Edge cases: [] never matches, and [^] and [!]
106  * always match without consuming a character.
107  *
108  * Note that these patterns attempt to match the entire string, not
109  * just find a substring matching the pattern.
110  *
111  * @param pattern The pattern to match to
112  * @param strings The string we are trying to match
113  * @param flags flags to use in the match. Bitwise OR of:
114  * <pre>
115  * APR_FNM_NOESCAPE Disable backslash escaping
116  * APR_FNM_PATHNAME Slash must be matched by slash
117  * APR_FNM_PERIOD Period must be matched by period
118  * APR_FNM_CASE_BLIND Compare characters case-insensitively.
119  * </pre>
120  */
121
